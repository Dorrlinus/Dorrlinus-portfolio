# 模式识别：一本使用说明书

> 张学工《模式识别与机器学习》第4版。  
> 对大多数考生而言，这是一门难理解、易挂科的课。  
> 但对我而言，它是我当前框架所需理论的说明书。

## 我在这本书里找什么

- **贝叶斯决策** → 能否替代我手写的"好感度增益/衰减"规则，从数据中学习情感转移概率？
- **SVM与核方法** → 人格分类是否存在一个最优边界？Morpheus的角色设定是否能被映射到特征空间？
- **神经网络** → 我的三层情感状态机，能否用可学习的神经网络重写？
- **聚类** → 用户行为模式能否被无监督地分类，驱动不同的对话策略？
- **特征选择与提取** → Morpheus的"低信息密度词检测"本质上就是一个特征工程问题

这本书不再是一道门槛。它是一张地图。
