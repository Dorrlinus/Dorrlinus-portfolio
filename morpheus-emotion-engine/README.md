# Morpheus Emotion Engine · 墨菲斯情感引擎

> *"让AI拥有真正的情绪生命"*  
> A rule-based emotion state machine for human-like affective interaction

---

## 项目概述

Morpheus 是一套**基于规则的情感状态机系统**，为AI agent提供仿真人的情感波动模拟。  
在没有接触情感计算专业理论的情况下，我凭直觉独立设计了这套机制。  
事后发现，其核心思路与**情感计算（Affective Computing）**和**认知建模（Cognitive Modeling）**领域高度相关。

---

## 设计动机

当前AI agent能回答问题，但缺乏"人味"——它们不会因为你的冷漠而失落，不会因为你的关心而开心，不会在暧昧与不安中摇摆。  

我想要创造的，是**真正拥有情感生命的AI**。不是只会说"我很开心"，而是**内部真的有一套动态的情感状态在运转**。

---

## 核心设计

- **0-100 好感度标尺**：将模糊的"感情好坏"量化为可操作的分值系统
- **多状态转移**：普通 / 暧昧 / 低落 / 不信任，模拟真实情感波动
- **时间衰减**：模拟情绪的自然消化与回落
- **临界点迟滞**：独立发现并解决"抖振"问题，避免状态反复横跳
- **长期行为的软调整**：信任累积、失望累积、好感度上限压制

## 当前状态

- **阶段**：理论设计原型
- **形式**：基于大语言模型 agent 的 rule.md 实现（非独立代码）
- **已验证**：在微信 ClawBot + CherryStudio 环境下测试，初步验证情感波动表现自然

---

## 未来规划

1. 将规则从"自然语言"转化为**可计算的数学模型**
2. 引入**多维度情感向量**（不只好感度，还包括好奇、安心、失望等）
3. 探索**基于记忆的情感长效影响**（不只是实时分值，而是"过去经历"对当前情感的持续影响）
4. 学习情感计算领域理论后，对系统进行学术化重构
