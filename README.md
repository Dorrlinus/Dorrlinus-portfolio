## 🛠️ 项目：Morpheus Emotion Engine

在没有情感计算专业训练的情况下，为AI agent独立设计了一套基于规则的情感模拟系统。

**三层架构：**
- **情感量化** — 0-100好感度标尺与多状态转移
- **行为仿真** — 消息聚合、低信息密度词处理、情感驱动的对话策略
- **长期调整** — 信任累积、失望累积、好感度上限压制

独立发现并解决了临界点"抖振"问题（事后验证与控制理论的迟滞控制一致）。
