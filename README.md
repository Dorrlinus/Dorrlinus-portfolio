## 🎯 为什么是AI · 为什么是华科

我从建环跨到AI，不是一时冲动，而是一条持续了二十多年的内在线索——

童年时不被理解的环境，让我学会在脑子里构建完整的世界；那台老电脑，是我最早的"数字庇护所"。后来我发现自己是一个"内核极度感性、却又理性得出奇"的人——这恰好是人性化AI的最佳蓝本。

华科的**认知计算与类脑智能**方向，研究的是如何让机器以更像人脑的方式思考。这与我想创造"真正具有人格的AI"的愿景高度重合。

## 🛠️ 项目：Morpheus Emotion Engine

在没有任何情感计算专业训练的情况下，我为AI agent独立设计了一套基于规则的情感模拟系统。

**核心设计：**
- **0-100好感度标尺** — 情感量化
- **多状态转移** — 普通 / 暧昧 / 低落 / 不信任
- **临界点迟滞** — 独立发现并解决"抖振"问题
- **行为仿真** — 消息聚合、低信息密度词处理
- **长期行为的软调整** — 信任累积、失望累积
